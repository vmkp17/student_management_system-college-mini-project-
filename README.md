# 🎓 Student Management System

A simple Student Management System developed using **Python, Flask, and MySQL** to manage student academic records.

---

## 📌 Features

- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- Automatic Grade Calculation
- Glassmorphism User Interface
- MySQL Database Integration

---

## 🛠 Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
